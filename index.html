You can access the app with window.app

View can be access using window.app.view

Model can be access using window.app.model

Some Oddities

  • Setting window.app.model.installed=false modifies the DOM, but not in the way you'd expect. It just removes the ="true" part of the checked="true" Which html still interprets as checked
  • Bindings appear to be 1-way. If you modify the anything input and call app.model.toJSON(), you'll notice the value hasn't changed

Questions
